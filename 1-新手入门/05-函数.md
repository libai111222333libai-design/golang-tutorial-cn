# 函数教程

## 什么是函数？
函数是一段可以重复使用的代码，它通常接受一些输入，进行处理，并可以返回一个输出。

## 定义函数
在 Go 语言中，可以使用 `func` 关键字来定义一个函数。以下是一个简单的函数示例：

```go
func add(a int, b int) int {
    return a + b
}
```

## 调用函数
可以通过直接使用函数名和括号来调用函数：

```go
result := add(1, 2)
fmt.Println(result) // 输出 3
```

## 函数参数
Go 语言支持多种类型的参数。可以定义不同数量和类型的参数。例如：

```go
func multiply(a int, b float64) float64 {
    return float64(a) * b
}
```

## 返回值
函数可以返回多个值：

```go
func divide(a int, b int) (int, int) {
    return a / b, a % b
}
```

## 总结
函数是 Go 语言中的重要组成部分，帮助开发者提高代码的复用性和可读性。
