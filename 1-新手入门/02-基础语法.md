# Go基本语法

## 变量声明
Go语言中的变量通过`var`关键字声明。可以同时声明多个变量：
```go
var a, b int
```
也可以使用简短声明方式：
```go
a := 10
b := 20
```

## 常量
常量通过`const`关键字声明，常量一旦声明不能修改：
```go
const PI = 3.14
```

## 数据类型
Go语言的基本数据类型包括：
- `int`：整数
- `float64`：浮点数
- `string`：字符串
- `bool`：布尔值

## 操作符
Go支持多种操作符，包括：
- 算术操作符：`+`、`-`、`*`、`/`、`%`
- 比较操作符：`==`、`!=`、`<`、`>`、`<=`、`>=`
- 逻辑操作符：`&&`、`||`、`!`

## 注释
Go支持单行注释和多行注释：
```go
// 单行注释
/* 多行注释 */
```

## 导入语句
使用`import`关键字导入包：
```go
import (
    "fmt"
)
```
