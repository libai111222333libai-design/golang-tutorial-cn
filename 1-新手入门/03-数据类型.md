# 数据类型

在 Go 语言中，有多种数据类型，每种数据类型都有其特定的用途和���势。在本教程中，我们将详细讨论 Go 的基本数据类型，并示范如何在实际编程中使用它们。

## 1. 布尔类型

布尔类型只包含两个值：true 和 false。它常用于条件语句中。

```go
var isActive bool = true
if isActive {
    fmt.Println("活动中")
} else {
    fmt.Println("未活动")
}
```

## 2. 整型

整型用于表示整数。Go 有多种整型，如 int、int8、int16、int32 和 int64。它们的大小和取值范围各不相同。

```go
var age int = 30
var distance int64 = 123456
```

## 3. 浮点型

浮点型用于表示有小数的数字，主要有 float32 和 float64 两种。

```go
var temperature float64 = 36.6
```

## 4. 字符串

Go 的字符串类型用于表示一串文本数据。字符串是不可变的。

```go
var name string = "小明"
```

## 5. 字符

字符类型用于表示单个字符，使用 rune 表示。

```go
var letter rune = 'A'
```

## 6. 数组

数组是固定大小的元素集合，所有元素必须是相同类型。

```go
var primes [5]int = [5]int{2, 3, 5, 7, 11}
```

## 7. 切片

切片是动态大小的数组，使用起来更加灵活。

```go
var slice []int = []int{1, 2, 3}
```

## 8. 映射

映射是一个无序的key-value对集合，类似于字典。

```go
var ages map[string]int = map[string]int{"小明": 30, "小红": 25}
```

## 9. 结构体

结构体是一种用户定义的数据类型，可以组合多个字段。

```go
type Person struct {
    Name string
    Age  int
}
var p Person = Person{Name: "小明", Age: 30}
```

## 结论

这些是 Go 语言中的基本数据类型。理解和掌握这些数据类型是编写高效 Go 代码的基础。
