# 控制流教程

控制流用于管理程序执行的顺序。一个程序可以根据条件判断来决定执行哪部分代码。控制流有几种类型，包括：

1. **条件语句**：根据条件执行不同的代码块。
   - `if` 语句
   - `switch` 语句

2. **循环语句**：重复执行代码块。
   - `for` 循环
   - `while` 循环

3. **跳转语句**：强制改变程序控制流。
   - `break` 语句
   - `continue` 语句
   - `goto` 语句

## 示例代码

### if 语句示例
```go
if x > 10 {
    fmt.Println("x 大于 10")
} else {
    fmt.Println("x 小于或等于 10")
}
```

### for 循环示例
```go
for i := 0; i < 5; i++ {
    fmt.Println(i)
}
```

## 总结
控制流是编程的核心，掌握控制流会使你能够编写出灵活且高效的程序。
