# 结构体和方法

在 Go 语言中，结构体是一种聚合数据类型，用于将多个相关的值组合在一起。结构体允许我们将不同的数据元素结合成一个单一的实体，方便地管理这些数据。

## 1. 定义结构体

结构体的定义使用 `type` 关键字，后面跟上结构体的名称和字段。例如：

```go
type Person struct {
	Name string
	Age  int
}
```

在这个例子中，我们定义了一个名为 `Person` 的结构体，包含两个字段：`Name`（字符串类型）和 `Age`（整型）。

## 2. 创建结构体实例

可以使用结构体字面量创建结构体的实例：

```go
p := Person{Name: "Alice", Age: 30}
```

## 3. 结构体方法

在 Go 中，结构体还可以拥有方法。方法是与特定类型（此例中为结构体）关联的函数。可以使用接收者（receiver）来定义方法：

```go
func (p Person) Greet() {
	fmt.Printf("你好，我是 %s，今年 %d 岁。\n", p.Name, p.Age)
}
```

要调用方法，使用实例：

```go
p.Greet()
```

综上所述，结构体和方法使得 Go 语言在面向对象编程方面具有更强的灵活性和可维护性。
